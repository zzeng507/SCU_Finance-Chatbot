# SCU Finance Chatbot

A university course project chatbot designed to answer finance-related questions using school-provided materials.

## Project Structure

- **frontend/**  
  User interface for interacting with the chatbot.

- **backend/**  
  Backend services including retrieval-augmented generation (RAG) logic and API handling.

## Features

- Document-based question answering  
- Retrieval-Augmented Generation (RAG) pipeline  
- Separation of frontend and backend for clarity and maintainability  

## Notes

- This project is developed for academic purposes.
- Sensitive information such as API keys is not included in the repository.
